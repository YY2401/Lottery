# Lottery 抽獎系統

瀏覽器端抽獎工具，支援獎品管理、機率控制、抽獎動畫與歷史紀錄。純靜態頁面，所有資料儲存在本地端。

## 功能

### 抽獎
- 單抽、五連抽、十連抽
- 加權隨機抽選，依機率與剩餘數量決定結果
- 獎品耗盡時自動將機率重新分配給剩餘獎品
- 抽獎結果支援三種顯示模式：名稱、圖片、兩者皆顯示
- 點擊結果圖片可放大檢視

### 獎品設置
- 新增獎品：上傳圖片（PNG/JPG）、設定機率、數量、顯示文字、文字顏色、背景顏色、顯示模式
- 編輯現有獎品的所有屬性（含更新圖片）
- 批次刪除選取的獎品
- 一鍵正規化機率至 100%
- 縮圖與放大圖尺寸可調整

### 歷史紀錄
- 自動記錄每次抽獎結果（抽獎者、獎品、時間）
- 關鍵字搜尋（抽獎者名稱或獎品名稱）
- 複製歷史紀錄到剪貼簿（CSV 格式）
- 匯出歷史紀錄為 Excel 檔案
- 清空紀錄（需確認）
- 最多保留 1000 筆紀錄

### 匯入 / 匯出
- 獎品設置匯出為 Excel（不含圖片）
- 從 Excel 匯入獎品設置（需包含 Name、Probability、Quantity 欄位）
- 歷史紀錄匯出為 Excel

### 測試模擬
- 模擬 100 次與 1000 次抽獎，顯示各獎品命中次數與比率分布表

## 使用方式

1. 直接在瀏覽器開啟 `index.html`
2. 輸入抽獎者名稱
3. 點擊抽獎按鈕
4. 點擊右上角「設置」管理獎品與機率

## 資料儲存

| 儲存位置 | 用途 |
|----------|------|
| IndexedDB (`lotteryDB` / `prizes`) | 獎品設定與圖片（base64） |
| localStorage (`lotteryHistory`) | 抽獎歷史紀錄 |
| localStorage (`thumbnailSize`, `enlargedSize`) | 圖片顯示尺寸設定 |

## 外部依賴

透過 CDN 載入，無需安裝：

- [SweetAlert2](https://sweetalert2.github.io/) — 對話框與彈窗 UI
- [SheetJS (XLSX)](https://sheetjs.com/) — Excel 匯入匯出

## 專案結構

```
index.html    — 頁面結構與 CDN 引入
styles.css    — 樣式
script.js     — 所有邏輯（抽獎、儲存、設定、匯入匯出、模擬）
images/       — 範例圖片
references/   — 第三方 skill 與函式庫參考
```
